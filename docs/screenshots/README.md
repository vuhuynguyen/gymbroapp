# Screenshots

Drop PNG/JPG screenshots here and they render in the root [`README.md`](../../README.md).

Expected filenames (referenced by the README's Screenshots grid):

| File | Screen |
|---|---|
| `login.png` | Auth — login / sign-up |
| `log.png` | Workout Log (home tab) |
| `live-session.png` | Live workout session |
| `plan.png` | Assigned plan (trainee) |
| `progress.png` | Progress & history |
| `profile.png` | Profile |

Tip: portrait device frames around 1080–1284 px wide look best; the README sizes them to 220 px.
Rename or add rows in the README grid to match whatever you capture.
